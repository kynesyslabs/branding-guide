Install

Import once, reference tokens everywhere. Plain CSS or Tailwind.

Plain CSS

<link rel="stylesheet" href="brand/tokens.css">
<link rel="stylesheet" href="brand/fonts.css">
<link rel="stylesheet" href="brand/components.css">

Tailwind

// tailwind.config.js
module.exports = {
  presets: [require('./brand/tailwind.preset.js')],
  content: ['./src/**/*.{html,js,ts,jsx,tsx}'],
};

Fonts load from Google Fonts CDN via fonts.css. For production, self-host with next/font or @fontsource.
